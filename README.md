# Elderly Rummy Match Game

A simplified, high-contrast web game designed for seniors with moderate dementia and hearing impairment.

## Features
- **Large Visuals:** Numbers are 70px+, optimized for aging eyes.
- **Deaf-Friendly:** 100% visual feedback (no audio required).
- **Rummikub Logic:** Uses the familiar "Same Number" or "Same Color" grouping rules.
- **Dementia-Focused UX:** Zero-fail mechanics; incorrect moves reset automatically without penalties.

## How to play
1. Click on tiles to move them to the bottom slots.
2. Match 3 tiles by **Number** (e.g., three 5s) or **Color** (e.g., three Blue tiles).
3. Clear the board to get a success message.

## Technical Details
- Built with **React** (via CDN for zero-setup deployment).
- Mobile-optimized (prevented zooming and scrolling).
- Hosted on GitHub Pages.
